== Description ==

Content Expire Scheduler to automatically expire Post and Pages after selected date, and display custom message to people of expiration.

This simple plugin allow you to set expiration dates for both post and pages. No extra configuration only simple and easy setting page for message, once page or post is expired this message replace actual content, but not remove content from post / page.

Every page and post have expiry calendar and custom message box and also support Custom Post Type.

= Features =
1. Easy date selection calendar
2. Set message for all expired posts and pages from Settings page.
3. Custom message for every seperate posts and pages.
4. Automatically disable comment box from expired content.
5. Support Custom Post Type.

= Note: = Posts / Pages content will not remove from back-end.

== Installation ==

1. Unzip the download package
2. Upload `content-expire-scheduler` to the `/wp-content/plugins/` directory
3. Activate the plugin through the 'Plugins' menu in WordPress

= Manual Plugin Installation =
1. Download Content Expire Scheduler Plugin to your desktop.
2. If downloaded as a zip archive, extract the Plugin folder to your desktop.
3. With your FTP program, upload the Plugin folder to the `wp-content/plugins` folder in your WordPress directory online.
4. Go to Plugins screen and find Content Expire Scheduler Plugin in the list.
5. Click Activate Plugin to activate it.

== Frequently Asked Questions ==

= Is this plugin removed expire content automatically =

No, this plugin not remove posts and page contents.

= Can I rescheduler the expired posts or pages =

Yes, easily you can reschedular the expire content from Post/Page Publish box.

= How can I hide the Expiry Date column from post or pages? =

Use the 'Screen Options' to show/hide Expiry Date column.

The 'Screen Options' pull down tab is located in the upper right corner of your Dashboard.
